# Hi, I'm Sam 👋

> **Building systems. Studying technology. Sharing what I learn.**

Welcome to my corner of GitHub.

I'm an engineer driven by curiosity and a desire to understand how technology works—from the infrastructure that powers modern applications to the human factors that shape how technology is built, trusted, and secured.

I enjoy building software, automating infrastructure, experimenting with new technologies, and documenting what I learn through research and writing. Whether I'm deploying cloud infrastructure, exploring AI, analyzing security concepts, or designing a new project, my goal is always the same: understand it deeply enough to explain it clearly.
